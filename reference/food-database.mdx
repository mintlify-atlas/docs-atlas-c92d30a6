---
title: "Food database"
description: "Complete reference for all food items tracked by Carbon Tracker, including emission factors and calculation details."
---

The food database contains every food item the app can track. Each item carries an **emission factor** expressed in kilograms of CO₂ equivalent per kilogram of food consumed (kg CO₂e/kg). These factors are derived from lifecycle assessment (LCA) research and account for production, processing, and transport.

## TypeScript interface

```typescript
export interface FoodItem {
  id: string
  name: string
  emoji: string
  emissionFactor: number // kg CO2e per kg of food
  category: string
}
```

<ResponseField name="id" type="string" required>
  Unique kebab-case identifier used in function calls (e.g. `"beef"`, `"ground-beef"`).
</ResponseField>

<ResponseField name="name" type="string" required>
  Human-readable display name shown in the UI.
</ResponseField>

<ResponseField name="emoji" type="string" required>
  Single emoji character representing the food item.
</ResponseField>

<ResponseField name="emissionFactor" type="number" required>
  Kilograms of CO₂ equivalent emitted per kilogram of this food. Used directly in emission calculations.
</ResponseField>

<ResponseField name="category" type="string" required>
  One of: `Meat`, `Seafood`, `Dairy`, `Grains`, `Legumes`, `Vegetables`, `Fruits`, `Beverages`, `Processed`.
</ResponseField>

## Emission calculation

Food emissions are calculated from the item's emission factor and the mass consumed:

```
emissions (kg CO₂e) = emissionFactor × (gramsConsumed / 1000)
```

### `calculateFoodEmissions(foodId, gramsConsumed)`

Returns the kg CO₂e produced by consuming a given mass of a food item.

```typescript
function calculateFoodEmissions(foodId: string, gramsConsumed: number): number
```

<ParamField path="foodId" type="string" required>
  The `id` field of the target `FoodItem`. Must match an entry in `FOOD_DATABASE`.
</ParamField>

<ParamField path="gramsConsumed" type="number" required>
  Mass of the food consumed, in grams.
</ParamField>

**Example**

```typescript
// A 250 g beef steak
calculateFoodEmissions("beef", 250)
// => 27.0 × (250 / 1000) = 6.75 kg CO₂e

// A 100 g serving of lentils
calculateFoodEmissions("lentils", 100)
// => 0.9 × (100 / 1000) = 0.09 kg CO₂e
```
